Pokemon Class¶

class Pokemon.Pokemon(type_list, stats, moves, name='unknown', level=1)¶

Class that defines the properties, attack and defense of the Pokemon

__init__(type_list, stats, moves, name='unknown', level=1)¶

Pokemon’s constructor

Parameters:
  • poke_type1 – Pokemon’s principal type
  • poke_type2 – Pokemon’s secondary type (default: none)
  • stats – Pokemon’s stats, object of the Stats class.
  • name – Pokemon’s name (for example, a Zubat’s name is ‘Batman’)
  • level – Pokemon’s level
  • moves – Pokemon’s moves, object of the MoveList class.
Returns:

A Pokemon instance with all attributes set
best_move(onPokemon)¶

Checks Pokemon’s best move (bigger damage) over the onPokemon

Parameters:onPokemon – The Pokemon’s opponent in the battle
Returns:The best move (bigger damage)
calculates_damage(move, onPokemon)¶

Calculates the damage of Pokemon’s attack over the onPokemon

Parameters:
  • move – Pokemon’s move that will be performed, object of the Move class.
  • onPokemon – The Pokemon’s opponent in the battle
Returns:

The damage received by the pokemon. If the pokemon missed the attack it returns -1
compare_types_to(pokemon)¶

Compare every combination between two pokemon’s types.

Parameters:pokemon – The deffensive pokemon.
Returns:The Type coefficient to be used on damage formula
expected_damage(move, onPokemon)¶

Returns the expected damage of Pokemon’s attack over the onPokemon

Parameters:
  • move – Pokemon’s move that will be performed, object of the Move class.
  • onPokemon – The Pokemon’s opponent in the battle
Returns:

The expected damage received by the pokemon.
is_alive()¶

Tests if the Pokemon is alive, i.e. if it’s hp is bigger than zero

Returns:True or False if the hp is bigger or less than zero
perform_move(move, onPokemon)¶

Pokemon tries to attack the onPokemon

Parameters:
  • move – Pokemon’s move that will be performed, object of the Move class.
  • onPokemon – The Pokemon’s opponent in the battle
Returns:

The damage received by the pokemon. If the pokemon missed the attack it returns -1
receive_damage(damage=0)¶

Decreases the hp of the Pokemon of a damage.

Parameters:damage – Value of damage that will be received
